# bcg-ops-governance
BCG Corp — Operational Governance Standards

This repository is the version-controlled authority for BCG's governance
documents, standards, and templates. All files here supersede any copies
maintained elsewhere (OneDrive, Claude project knowledge bases, etc.).

Contents: Governance Doc Registry, BCG Style Guide, Initiative & Workstream
Catalog, Ecosystem Awareness Block master, SOP templates, artifact naming
and capture standard, project ecosystem and handoffs reference.
